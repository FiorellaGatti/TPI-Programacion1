# TPI-Programacion1
# Sistema de Gestión de Datos de Países

## Descripción
Aplicación desarrollada en Python que permite gestionar y analizar información demográfica de países del mundo. El sistema opera mediante un menú interactivo en consola y trabaja con datos almacenados en un archivo CSV.

- Lautaro Gatti

## Requisitos
- Python 3.x
- No requiere instalación de librerías externas

## Instrucciones de uso
1. Clonar el repositorio
2. Asegurarse de que el archivo `paises.csv` se encuentra en la misma carpeta que `main.py`
3. Ejecutar el programa desde la terminal con el comando: python main.py

## Opciones del menú
1. **Agregar un país** — permite ingresar un nuevo país con todos sus datos
2. **Actualizar datos de un país** — modifica la población y superficie de un país existente
3. **Buscar un país** — busca por nombre con coincidencia parcial o exacta
4. **Filtrar por continente** — muestra los países de un continente específico
5. **Filtrar por rango** — filtra países por rango de población o superficie
6. **Ordenar países** — ordena por nombre, población o superficie (ascendente o descendente)
7. **Mostrar estadísticas** — muestra mayor/menor población, promedios y cantidad por continente
8. **Salir** — cierra el programa
